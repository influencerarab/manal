# موقع منال ومشاري

موقع صفحة واحدة جاهز للرفع على GitHub Pages.

## طريقة الرفع
1. ارفع كل الملفات داخل المستودع.
2. من Settings > Pages اختر Branch: main و Folder: /root.
3. احفظ الإعدادات وانتظر رابط الموقع.

## تعديل الصور
استبدل الصور داخل:
assets/images
مع الحفاظ على نفس الأسماء إن أردت ظهورها تلقائيًا.
