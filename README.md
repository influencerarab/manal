# Manal & Meshari Media Kit Website

موقع صفحة واحدة جاهز لـ GitHub Pages.

## طريقة الرفع على GitHub Pages
1. افتح GitHub وأنشئ Repository جديد.
2. ارفع الملفات: `index.html` ومجلد `assets`.
3. من Settings > Pages اختر Branch: `main` ثم `/root`.
4. سيظهر رابط الموقع خلال دقائق.

## تغيير الصور
استبدل مكان `photo-placeholder` في `index.html` بصورة حقيقية، أو أرسل الصور ليتم إدخالها داخل التصميم.

## تعديل الأرقام
الأرقام الحالية موضوعة كـ placeholders مثل `+0M` حتى تزودني بالأرقام الحقيقية.
